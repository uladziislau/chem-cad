# Вектор развития: Digital Formulation Lab (Химический САПР)

## 🎯 Глобальная цель
Создание браузерной Системы Автоматизированного Проектирования (САПР) для химиков-технологов. Инструмент предназначен для проектирования, моделирования и производства реальных химических продуктов передового уровня (косметики, полимеров, лекарственных форм, удобрений, пищевых добавок и т.д.).

## 🧬 Ключевая философия
* **Local-First & High-Performance:** Максимальное использование вычислительных мощностей устройства пользователя (Apple Silicon M1, WebGPU, WebGL) без жесткой привязки к облачным бэкендам (Firebase). Данные живут в оперативной памяти сессии.
* **Инженерный фокус:** Переход от "просто красивой 3D-картинки" к практическим расчетам, смешиванию и анализу.
* **Единое рабочее пространство:** Отказ от разрозненных страниц в пользу единого мощного дашборда (Lab Dashboard), где все инструменты находятся под рукой.

## ⚙️ Конвейер производства (Pipeline)
Разработка строится вокруг трех уровней проектирования продукта:

1. **Микро-уровень (Синтез и Анализ Ингредиентов):**
   * Ввод химической структуры (SMILES).
   * Парсинг в топологические графы (ООП-модель).
   * 3D-фолдинг и минимизация энергии в реальном времени (WebGPU/Three.js).
   * Расчет базовых свойств (молярная масса, брутто-формула).

2. **Макро-уровень (Формуляция / Блендинг):**
   * Создание смесей из спроектированных молекул (ингредиентов).
   * Управление массовыми/объемными долями (процентами) через UI.
   * Расчет итоговых свойств композиции (средняя молярная масса, плотность и т.д.).

3. **Уровень валидации (Термодинамика и Безопасность):**
   * Проверка совместимости ингредиентов (в будущем).
   * Предсказание стабильности смеси.

## 🚀 Ближайшие шаги (Roadmap)
1. **Внедрение модуля Formulator (Смеситель):** Добавление в Lab Dashboard вкладок для переключения между просмотром одной молекулы и сборкой рецептуры (смеси).
2. **Управление пропорциями:** UI для настройки процентов каждого ингредиента в итоговом продукте.
3. **Экспорт рецептуры:** Возможность выгрузить готовую формулу (JSON/PDF/CSV) для передачи на реальное производство.
