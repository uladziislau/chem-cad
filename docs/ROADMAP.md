# Дорожная карта развития проекта ChemCraft (Roadmap)

## Текущее состояние
- [x] Базовая архитектура (Angular 21, Tailwind CSS)
- [x] Дашборд и навигация
- [x] Учебный класс (базовый)
- [x] Виртуальная лаборатория (базовая)
- [x] 3D Моделирование (SmilesDrawer, 3Dmol.js, PubChem API)
- [x] База рецептур
- [x] Формулатор
- [x] Раздел безопасности

## Направления для дальнейшего развития

### 1. Интеграция ИИ (AI Assistant)
- **ИИ-Химик (Gemini):** Интеграция `@google/genai` для создания умного ассистента.
- **Анализ рецептур:** ИИ анализирует созданную в "Формулаторе" рецептуру на предмет совместимости компонентов, стабильности и безопасности.
- **Генерация идей:** ИИ предлагает новые формулы по запросу (например, "создай рецепт экологичного шампуня без сульфатов").

### 2. Продвинутый Формулатор (Pro Formulator)
- **Расчет HLB (ГЛБ):** Автоматический расчет гидрофильно-липофильного баланса для создания стабильных эмульсий (кремы, лосьоны).
- **Прогнозирование pH:** Оценка итогового pH продукта на основе кислотности/основности компонентов.
- **Экономика:** Калькулятор себестоимости продукта на основе цен сырья.
- **Генерация SDS:** Автоматическое формирование паспортов безопасности (Safety Data Sheet) в PDF.

### 3. Синергия компонентов (Ecosystem Synergy)
- **Связь Лаборатории и 3D:** При успешном проведении реакции в "Виртуальной лаборатории" автоматически генерировать SMILES продукта и открывать его в 3D-просмотрщике.
- **База данных реакций:** Расширение движка реакций для поддержки органического синтеза (этерификация, омыление и т.д.).

### 4. Облако и Сообщество (Cloud & Community)
- **Firebase Интеграция:** Подключение базы данных Firestore и аутентификации.
- **Личный кабинет:** Сохранение пользовательских рецептур, истории экспериментов и любимых молекул.
- **Социальные функции:** Возможность делиться своими рецептами с другими пользователями, ставить оценки и оставлять комментарии.

### 5. Геймификация обучения (Gamification)
- **Интерактивные квесты:** Задания в "Учебном классе", где нужно синтезировать определенное вещество, чтобы пройти уровень.
- **Система достижений:** Награды за создание первого продукта, успешное прохождение тестов по безопасности и т.д.
