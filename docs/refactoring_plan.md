# План рефакторинга ХимСАПР и ChemCore

## 1. Реорганизация структуры доменов
Текущая структура будет приведена к более строгому разделению ответственности:
- `architecture/models`: Только чистые интерфейсы и базовые классы (Data Models).
- `architecture/engines`: Специализированные вычислительные движки (ChemCore, WebGPU, Thermodynamics).
- `architecture/calculators`: Математические функции для расчета конкретных свойств (LogP, TPSA).

## 2. Улучшение ChemCore (Custom Engine)
- **Переход на асинхронность:** Все тяжелые расчеты свойств теперь `async`, что позволяет внедрять GPU-ускорение без блокировки UI.
- **Интеграция WebGPU:** Использование `WebGpuEngineService` для параллельных вычислений.
- **Расширение моделей:** Добавление поддержки ароматичности и гибридизации на уровне графа.

## 3. WebGPU Шейдеры (WGSL)
- **Force Field Shader:** Реализация расчета сил межатомного взаимодействия (Lennard-Jones, Coulomb) на GPU.
- **Conformer Search:** Параллельный поиск стабильных конформаций молекулы.

## 4. UI/UX Оптимизация
- **Real-time Feedback:** Отображение статуса GPU в интерфейсе.
- **Системные логи:** Более детальное логирование этапов работы движка (Topology -> Physics -> GPU).
